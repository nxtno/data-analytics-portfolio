## E-Commerce BI Analysis

**Business Question:** *What drives revenue — and how can we reduce delivery failures?*

A full-stack BI project built end-to-end: from raw data through Python EDA and SQL analytics to a published Tableau story.

| Layer | Tool | What I did |
|-------|------|------------|
| Exploration | Python · pandas · matplotlib · seaborn | EDA, cleaning, statistical analysis |
| Analytics | PostgreSQL · SQL | Window functions, CTEs, cohort analysis, LTV segmentation |
| Visualization | Tableau Public | 3-chapter interactive story dashboard |

**Key Findings:**
- $47,728 total revenue — but $2,349 (4.9%) lost to failed deliveries
- Electronics (Feb 2023): 50% failure rate — worst single-month result across all categories
- Top 20% of customers generate ~44% of revenue
- January cohort: sharpest retention drop from month 0 → 1 (100% → 10.3%)
